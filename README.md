# Interface-Funcional-EBAC
# Filtro de Nomes com Lambda e Functional Interface

Exercício em Java demonstrando o uso de:
- Expressões Lambda
- `@FunctionalInterface`
- Stream API

## Funcionalidades

- Cria uma lista de nomes
- Filtra nomes que começam com a letra "A"
- Utiliza interface funcional personalizada `FiltroNome`

## Tecnologias

- Java 8+

## Como executar

```bash
javac src/FiltroNomesLambda.java
java -cp src FiltroNomesLambda
